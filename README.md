# Scrabble

Jeu de Scrabble en ligne jouable dans le navigateur.

Le fichier principal du site est `index.html`, prêt pour GitHub Pages.

## Multijoueur avec Supabase

1. Crée un projet Supabase.
2. Ouvre `SQL Editor` dans Supabase.
3. Exécute le contenu de `supabase-schema.sql`.
4. Dans `scrabble.html`, remplace :
   - `REMPLACE_PAR_TON_SUPABASE_URL`
   - `REMPLACE_PAR_TON_SUPABASE_ANON_KEY`
5. Copie ensuite `scrabble.html` vers `index.html`, puis redéploie.
